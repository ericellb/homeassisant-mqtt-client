### Installation

1. Clone the repo

   ```
   git clone https://github.com/ericellb/homeassisant-mqtt-client
   ```

2. Install deps
   ```
   yarn install OR npm install
   ```
3. Create your .env file
   ```ts
   // EXAMPLE ENV FILE
   MQTT_URL = 'mqtt://192.168.0.3:1883';
   MQTT_USERNAME = 'your_username';
   MQTT_PASSWORD = 'your_password';
   ```
